## 项目简介
通过HTTP请求，调用图灵机器人API，拿到POST请求的数据，结合内置命令行操作

## 运行配置
1. `npm start` 启动程序
2. `npm link` 映射全局，调用全局命令`robot`即可运行聊天程序
3. `npm publish` 发布
